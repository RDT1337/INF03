łączenie jedej bazy z drugą:


          JOIN druga_baza ON
          druga_baza.klucz_obcy/główny = pierwsza_baza.klucz_obcy/główny

łączenie baz nie połączonych bezpośrednio

łączysz wtedy ze sobą te bazy idąc przez pośrednie bazy
czyli: BAZA_START -> BAZA2 -> BAZA3 -> ... -> BAZA_DOCELOWA
łączysz je za pomocą złączeń które mają ze sobą te poszczególne bazy
