Rodzaje znaczników zmiennych w JS:

  • VAR
    • Pozwala na bycie użytym poza swoim kontenerem
    • Strarszy rodzaj zmiennej, rzadziej używany
  • LET
    • Dynamiczny typ zmiennej
    • Nie użyjesz go poza klamrami
  • CONST
    • Nie można używać go w wyższych klamrach

Rodzaje danych w zmiennych w JS:

  • Number
  • Bigint
  • String
  • Boolean
  • Undefined
  • Null
  • Symbol
  • Object
