WYJAŚNIENIE DZIAŁU PHP I MYSQL

PHP i MYSQL nie jest funkcjonujące na github bo do działania tych języków wymagane jest połączenie lokalne, czego github nie wspiera.
dlatego tutaj będzie tylko wyświetlanie kodu, wyjaśnienie funkcjonalności, szablony do skopiowania na urządzenie lokalne itp.

  1. Połączenie do bazy
  2. SELECT z bazy do strony HTML
  3. INSTERT INTO z HTML do bazy
  4. DELETE FROM z HTML do bazy
  5. Wyświetlanie danych z formularza HTML w postaci tabeli
  6. Odpięcie połączenia z bazą
